# Knowledge Priming

> **Excerpt from the original:** AI coding assistants default to generic patterns from their training data. I propose treating project context as infrastructure—versioned files that prime the model before each session—rather than relying on ad-hoc copy-pasting. This is essentially manual RAG (Retrieval-Augmented Generation), and I believe it fundamentally changes the quality of AI-generated code.
