# NVIDIA NeMo Guardrails: Programmable guardrails for LLM systems

> **Excerpt from the original:** "An open-source toolkit for easily adding programmable guardrails to LLM-based conversational systems." NeMo Guardrails (vendor: NVIDIA) enforces constraints via five rail categories — **input**, **dialog**, **retrieval**, **execution** (applied to the input/output of custom actions, a.k.a. the tools the LLM calls), and **output** rails.

**Relevance to constraints-driven development:** A runtime guardrail/policy engine for bounding agent behaviour; the execution-rail coverage of tool calls makes it applicable to coding-agent loops. License: Apache-2.0. Caveat: framed primarily for conversational systems, not coding-agent-specific.
