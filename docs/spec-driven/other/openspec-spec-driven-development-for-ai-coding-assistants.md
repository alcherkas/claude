# OpenSpec: Spec-driven development for AI coding assistants

> **Excerpt from the original:** "Spec-driven development (SDD) for AI coding assistants." OpenSpec is a lightweight, open-source SDD framework (site: openspec.dev) published by the Fission-AI org and installed via npm (`npm install -g @fission-ai/openspec@latest`). It drives the workflow through slash commands such as `/opsx:explore` and `/opsx:propose`, creating change proposals and specs that coding agents implement.

**Relevance to spec-driven development:** An emerging, org-published (not major-vendor) alternative to GitHub Spec Kit and Amazon Kiro — a tool-agnostic way to make specifications the shared source of truth across AI coding assistants. License: MIT.
