# Guardrails AI: Input/output validation framework for LLM applications

> **Excerpt from the original:** "Guardrails is a Python framework that helps build reliable AI applications by performing two key functions" — input/output validation and structured data generation. It runs **Input/Output Guards** that "detect, quantify and mitigate the presence of specific types of risks," built from 65+ Guardrails Hub validators (toxicity, PII, hallucination, …) and Pydantic/RAIL schema validation.

**Relevance to constraints-driven development:** A guardrail/policy engine plus schema validator for constraining and structuring LLM output. License: Apache-2.0. Caveat: a general LLM-application framework, not coding-agent-specific.
