# Qodo-Cover: AI agent for automated test generation and coverage

> **Excerpt from the original:** "An AI-Powered Tool for Automated Test Generation and Code Coverage Enhancement." Qodo-Cover (vendor: Qodo-AI; formerly CodiumAI Cover-Agent / `cover-agent`, itself based on Meta's TestGen-LLM) wires deterministic feedback sensors into an agent loop: a **Test Runner** executes the suite and produces coverage reports, and a **Coverage Parser** validates that coverage increases before generated tests are accepted — an iterative generate → run → validate → repeat cycle.

**Relevance to TDD / feedback-sensor harnesses:** A representative example of wiring deterministic gates (test runner + coverage parser) into an autonomous test-generation loop so only passing, coverage-increasing tests survive.
