# Beyond Vibe Coding: Amazon Introduces Kiro, the Spec-Driven Agentic AI IDE

InfoQ, Aug 2025

> **Excerpt from the original:** AWS recently released Kiro, a new VS Code fork aimed at taking developers beyond vibe coding and remedying some of its downsides. Kiro directly supports spec-driven development. Developers describe their requirements in natural language. Kiro outputs user stories with their acceptance criteria, a technical design document, and a list of coding tasks implementing the requirements. After review, developers may trigger the tasks implementation one step at a time.
