# Spec-driven development | Technology Radar | Thoughtworks

Radar technique blip — "Assess" ring

> **Excerpt from the original:** Spec-driven development is an emerging approach to AI-assisted coding workflows. While the term's definition is still evolving, it generally refers to workflows that begin with a structured functional specification, then proceed through multiple steps to break it down into smaller pieces, solutions and tasks. The specification can take many forms: a single document, a set of documents or structured artifacts that capture different functional aspects.
