# Microsoft Defender: AI agent runtime protection & pre-tool-call blocking

> **Excerpt from the original:** Real-time runtime protection that inspects the agent loop — user prompts, pre-tool calls, and post-tool responses — and can audit or block unsafe tool calls before they execute. For cloud agents it integrates with Work IQ MCP to evaluate tool invocations pre-execution; for local endpoint agents such as Claude Code and GitHub Copilot CLI it uses provider-supported application-layer hooks. Block / Audit / Disabled policy modes. Defender also correlates agent alerts into incidents with an incident graph for blast-radius investigation.

**License:** Microsoft product (Preview as of June 2026; Agent 365 licensing; a 1 July 2026 subscription requirement for cloud-agent blocking)

**Relevance to securing & governing AI coding agents:** The most complete enterprise runtime kill/contain control surfaced — pre-tool-call interception that can block a risky action before execution, plus incident correlation for blast-radius understanding. Almost entirely Preview; behavior and availability may change.
