# MCP-Scan / Snyk Agent Scan: Security scanner for MCP servers & agents

> **Excerpt from the original:** Security scanner for AI agents, MCP servers and agent skills. For MCP, it connects to servers and retrieves tool descriptions, then scans them for common threats like prompt injections, sensitive data handling, or malware payloads hidden in natural language — detecting 15+ risk types including Tool Poisoning, Tool Shadowing and Toxic Flows. Now published by Snyk as snyk-agent-scan after the Invariant Labs acquisition.
