# Arcade.dev: OAuth tool authorization for AI agents

> **Excerpt from the original:** Each tool in Arcade's MCP servers has a set of required permissions — or, more commonly referred to in OAuth 2.0, scopes. A scope is what the user has authorized someone else (in this case, the AI agent) to do on their behalf. This gives fine-grained control over what data third-party services can access and what actions they can take (e.g. Gmail.SendEmail requires only the gmail.send scope).

**License:** Commercial / vendor product

**Relevance to securing & governing AI coding agents:** Runtime/authorization layer where each agent tool declares its required OAuth 2.0 scopes, enforcing user-delegated, least-privilege, just-in-time access on a user's behalf instead of broad service accounts.
