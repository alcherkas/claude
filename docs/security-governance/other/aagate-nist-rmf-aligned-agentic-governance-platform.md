# AAGATE: A NIST AI RMF-aligned governance platform for agentic AI

> **Excerpt from the original:** AAGATE assigns a named security framework per NIST RMF function, LLM-translates natural-language regulations (EU AI Act, ISO 42001, internal red-lines) into machine-readable Open Policy Agent Rego policy, and uses a Janus Shadow-Monitor-Agent to evaluate an agent's planned actions in real time before harm — replacing periodic manual red teaming with continuous in-loop red teaming.

**License:** arXiv preprint (single source, medium confidence)

**Relevance to securing & governing AI coding agents:** Research-stage platform that operationalizes the NIST AI RMF with a named framework per function (CSA MAESTRO for Map; AIVSS+SSVC for Measure; CSA Agentic Red Teaming for Manage), guardrails-as-code (NL→Rego/OPA), and the Janus shadow-monitor plan-then-execute pattern. Design, not measured efficacy.
