# Open Policy Agent (OPA/Rego) & OPAL: general-purpose policy enforcement

> **Excerpt from the original:** OPA is a general-purpose policy engine that decouples policy decisions from application logic, deployed as a sidecar, standalone daemon, or embedded Go library evaluating Rego policies against incoming requests as a Policy Decision Point (PDP). OPAL (Open Policy Administration Layer) is an administration layer for OPA and AWS Cedar Agent that detects policy/data changes in real time and pushes live updates over a websocket PubSub channel.

**License:** OPA: Apache-2.0 (CNCF) · OPAL: MIT (permitio/opal)

**Relevance to securing & governing AI coding agents:** The foundational, production-grade enforcement primitive under agent-governance products (Permit.io, the Microsoft toolkit, TrueFoundry). Mature for authorization, but not agent-specific — agent-action policies are hand-authored in Rego.
