# Design patterns for securing LLM agents against prompt injection (lethal trifecta, dual-LLM)

> **Excerpt from the original:** Six design patterns for building LLM agents that resist prompt injection — including the Dual LLM pattern (a privileged LLM never sees untrusted content; a quarantined LLM processes it without tool access) — and the 'lethal trifecta': an agent with access to private data, exposure to untrusted content, and the ability to externally communicate is exploitable.
