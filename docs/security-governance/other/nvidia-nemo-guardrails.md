# NVIDIA NeMo Guardrails: programmable runtime rails for LLM systems

> **Excerpt from the original:** An open-source toolkit adding programmable guardrails to LLM conversational systems, enforced at runtime via a dialogue-management-inspired runtime, the Colang DSL (.co files), and five rail types — input, dialog, retrieval, execution, and output — applied at different stages. Controls include topic restriction, jailbreak detection, and output formatting; execution rails gate tool/action calls.

**License:** Apache-2.0

**Relevance to securing & governing AI coding agents:** Content/behavioral guardrailing, not authorization. Execution rails are the most agent-relevant surface for gating tool calls, but the toolkit targets conversational LLMs generally rather than coding agents specifically. Colang 2.0 is beta with 1.0 still default.
