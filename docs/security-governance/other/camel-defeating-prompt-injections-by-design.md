# CaMeL: Defeating prompt injections by design

> **Excerpt from the original:** CaMeL (CApabilities for MachinE Learning) is a defense that treats the LLM as untrusted: it derives an explicit control- and data-flow plan from the user's trusted query and enforces capability-based policies on tool calls, so untrusted content cannot redirect actions — defeating prompt injection by construction rather than by detection.
