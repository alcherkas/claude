# Auth0 for AI Agents: Token Vault & human-in-the-loop approval

> **Excerpt from the original:** Using Token Vault, AI agents can fetch access tokens for external identity providers to call their APIs, all through a single Auth0 integration. Asynchronous authorization implements the Client-Initiated Backchannel Authentication (CIBA) standard, enhanced with Rich Authorization Requests (RAR) — even if an AI agent is compromised, it can't execute sensitive actions without explicit human approval.
