# LlamaFirewall: Open-source guardrail framework for AI agents

> **Excerpt from the original:** A framework designed to detect and mitigate AI-centric security risks, supporting multiple layers of inputs and outputs, such as typical LLM chat and more advanced multi-step agentic operations. Three guardrails ship in the box: PromptGuard 2, Agent Alignment Checks, and CodeShield. See Meta's research paper 'LlamaFirewall: An open source guardrail system for building secure AI agents' (arXiv:2505.03574).
