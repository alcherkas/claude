# OpenAI Swarm: Lightweight multi-agent orchestration (educational)

> **Excerpt from the original:** An educational framework exploring ergonomic, lightweight multi-agent orchestration. Swarm is not intended to be used in production, and therefore has no official support. It explores patterns that are lightweight, scalable, and highly customizable by design — built around two primitives: Agents and handoffs.

**Relevance to agentic orchestration & workflows:** OpenAI's experimental predecessor to the Agents SDK. It introduced the **routines + handoffs** model — stateless agents that pass control to one another — that was later productionized in the OpenAI Agents SDK. Included here for lineage and as the minimal reference design for handoff-based orchestration. *Maturity: primary repo, explicitly educational / not for production (backfill).*
