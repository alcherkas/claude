# Don't Build Multi-Agents

> **Excerpt from the original:** Principle 1: Share context, and share full agent traces, not just individual messages. Principle 2: Actions carry implicit decisions, and conflicting decisions carry bad results. [The post argues for single-threaded, continuous-context agents over parallel multi-agent architectures for coding work, because subagents that don't share full context make conflicting implicit decisions that are hard to reconcile.]
