# Inngest: Durable execution for AI agents

> **Excerpt from the original:** Durable execution lets you break an agent's work into discrete steps that are individually checkpointed and automatically retried; completed steps are memoized so a function that crashes resumes from the last successful step rather than re-running everything — turning unreliable, long-running LLM calls and tool invocations into fault-tolerant workflows.
