# Orchestrate agent behavior with generative AI - Microsoft Copilot Studio

Copilot Studio planner

> **Excerpt from the original:** Agents can use either generative or classic orchestration. By default, newly created agents use generative orchestration. An agent that uses generative orchestration can choose the best tools, knowledge, topics, and other agents to answer user queries or respond to event triggers. The alternative is classic orchestration, where an agent responds to users by triggering the topic whose trigger phrases most closely match the user's query.
