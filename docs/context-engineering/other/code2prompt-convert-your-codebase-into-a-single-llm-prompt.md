# code2prompt: Convert your codebase into a single LLM prompt

> **Excerpt from the original:** A CLI tool (Rust core) to convert your codebase into a single LLM prompt with a source tree, prompt templating (Handlebars), and token counting. Traverses directories with `.gitignore`/glob filters. Beyond CLI packing it ships an MCP server ("run code2prompt as a local service, enabling agentic applications to read your local codebase efficiently") and fast Python bindings for RAG/agent pipelines.

**Relevance to multi-repo context engineering:** A directly comparable alternative to Repomix in the packing category — turns a repo into one token-counted prompt — but additionally exposes an MCP server and Python SDK, bridging the packing approach into agent/RAG workflows. Note: the MCP server delivers packed output rather than a semantic index.
