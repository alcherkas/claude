# Serena: Semantic MCP toolkit for coding agents

> **Excerpt from the original:** A powerful MCP toolkit for coding, providing semantic retrieval and editing capabilities — operating at the symbol level and exploiting relational structure rather than line numbers or primitive search patterns. Integrates with any client/LLM via the Model Context Protocol and incorporates an abstraction layer over language servers (LSP) for 40+ programming languages. Connects to Claude Code, Codex, OpenCode, Gemini-CLI, VS Code/Cursor/JetBrains, and Claude Desktop. Free and open-source (MIT).

**Relevance to multi-repo context engineering:** An indexing/semantic approach (distinct from packing) that serves targeted, symbol-level context on demand via MCP — letting an agent retrieve exactly the symbols and relationships it needs instead of a whole-repo dump. Scope is per-repo.
