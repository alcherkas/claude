# CodeGraph: Pre-indexed knowledge graph for AI code context

> **Excerpt from the original:** Surgical context · fewer tool calls · faster answers · 100% local. CodeGraph builds a local SQLite index of symbols, dependencies, and call paths so AI agents get surgical code context in a single tool call instead of slow file-by-file exploration. A query returns the relevant symbols' verbatim source grouped by file, plus the call paths between them and a blast-radius summary — reducing tool invocations by ~58% on average and cutting token usage on larger codebases. Works with Claude Code, Cursor, and other agents.

**Relevance to multi-repo context engineering:** Replaces grep/read exploration with a pre-built, queryable code knowledge graph that returns just-in-time, dependency-aware context — directly addressing the token-budget and retrieval cost of giving agents context across large or multiple codebases.
