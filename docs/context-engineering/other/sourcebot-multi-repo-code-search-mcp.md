# Sourcebot: multi-repo code search with an MCP server

> **Excerpt from the original:** Sourcebot uses **Zoekt under the hood** for multi-repo regex search across GitHub/GitLab/Bitbucket/Gitea/Gerrit, adds goto-definition / find-references and a natural-language chat layer, and ships a **production MCP server** (`@sourcebot/mcp`). It deploys as a single Docker Compose stack and is actively developed (v5.x, mid-2026).

**Relevance to multi-repo context engineering:** One of the **few production-grade multi-repo MCP code-search servers** — i.e. a turnkey way to give an agent cross-repo search without building your own Zoekt wrapper. Licensing caveat (verified this pass): Sourcebot is **source-available under a Fair Source license, NOT OSI-approved open source** — do not describe it as "OSS." It is *among* the few such servers, not provably "the only" one.
