# Repomix: Pack your codebase into AI-friendly formats

> **Excerpt from the original:** Repomix is a command-line tool that packs your entire repository into a single, AI-friendly file. Build AI-optimized output (XML, Markdown, JSON, or plain text) with token counting, Tree-sitter–based compression (~70% token reduction), remote-repo support, `.gitignore`/`.repomixignore` filtering, Secretlint security scanning, and an MCP server so assistants like Claude can pull a packed codebase directly.

**Relevance to multi-repo context engineering:** Packs one or more repositories into a single token-optimized artifact you can feed an agent, and runs as an MCP server — a practical way to give an agent whole-repo context (and, by packing several, cross-repo context) within an LLM's context window.
