# Augment Code — Context Engine MCP: cross-repo context for any agent

> **Excerpt from the original:** Augment's Context Engine maintains a persistent semantic index plus a cross-repo dependency graph spanning **entire orgs across GitHub/GitLab/Bitbucket**, now exposed as a generally-available hosted **MCP server** usable from any agent (Claude Code, Cursor, Codex). The **remote/hosted mode is the one that delivers cross-repo** context; a GitHub App connection keeps the index commit-fresh.

**Relevance to multi-repo context engineering:** The strongest *dedicated, agent-facing* cross-repo option found in the June 2026 research pass — a real semantic index + dependency graph over whole organizations, served to any MCP client. Caveats: it is a **gated SaaS** (proprietary), though it offers VPC-isolated / air-gapped deployment, CMEK, and SOC 2 Type II; and every headline performance figure (e.g. "70%+ agent quality gains," SWE-bench scores) is **vendor self-reported and unreplicated** — treat the percentages as marketing, not benchmarks.
