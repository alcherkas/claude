# Inspect: UK AISI framework for LLM & agent evals

> **Excerpt from the original:** `inspect_ai` is a framework for large language model / frontier AI evaluations, created by the **UK AI Security Institute (AISI)** in collaboration with Meridian Labs.

**License:** MIT (open-source).

**Integration / coding-agent relevance:** **Built for coding and agentic tasks** (alongside reasoning, knowledge, behavior, and multimodal), not just general LLM apps. Supports **MCP tools** and built-in tools, and can run **external coding agents — Claude Code, Codex CLI, Gemini CLI — with Docker/Kubernetes sandboxing**. One of the most genuinely coding-agent-targeted eval frameworks in this catalogue. See also [Inspect Evals](inspect-evals.md) for the benchmark suite built on it.
