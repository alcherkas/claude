# Weights & Biases Weave: tracing & evaluation toolkit

> **Excerpt from the original:** Weave is an open-source toolkit for AI application development that logs and debugs LLM inputs, outputs, and traces, builds rigorous evaluations, and organizes information across the LLM workflow from experimentation to production.

**License:** Apache-2.0 (open-source). Separately-priced W&B cloud offering exists.

**Integration / coding-agent relevance:** Python/TypeScript SDK; accepts **OpenTelemetry** trace data via a dedicated **OTLP HTTP endpoint**. General LLM/agent observability, not coding-agent-specific.
