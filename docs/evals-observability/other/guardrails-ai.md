# Guardrails AI

**License:** Apache-2.0 (open-source).

**Integration / coding-agent relevance:** A runtime guardrail/validation library for the quality-gate bucket — wrap LLM calls so outputs must pass structural/safety/format validators before they propagate. General LLM tooling; applicable to coding agents as an output-validation layer.
