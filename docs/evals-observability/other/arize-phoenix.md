# Arize Phoenix: open AI observability & evaluation

> **Excerpt from the original:** "Phoenix is an open-source AI observability platform designed for experimentation, evaluation, and troubleshooting" of LLM applications.

**License:** **Elastic License 2.0 (ELv2)** — *source-available, NOT OSI-approved.* Free to self-host with no feature gates, but bars offering Phoenix as a hosted/managed service, so the common "open-source" label is technically loose.

**Integration / coding-agent relevance:** OpenTelemetry-based instrumentation via the **OpenInference** project; direct Python (`arize-phoenix-otel`/`-client`/`-evals`) and TypeScript packages; an **MCP server** (`@arizeai/phoenix-mcp`) plus coding-agent skills usable from Claude Code and Cursor *(the skills are a separate repo component, not bundled in the npm MCP package — a 2-1 verification nuance)*. Targets both general LLM apps and coding agents.
