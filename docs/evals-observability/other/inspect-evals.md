# Inspect Evals: community benchmark suite on Inspect

> **Excerpt from the original:** A repository of dozens of community-contributed LLM benchmark evaluations built on Inspect AI — by the UK AI Security Institute with Arcadia Impact and the Vector Institute — for model capability and safety benchmarking, including coding-agent benchmarks like SWE-Bench plus GAIA, GDM CTF, and Cybench.

**License:** MIT (open-source).

**Integration / coding-agent relevance:** Capability/safety **benchmarking**, not production observability. The coding-agent benchmarks (e.g., [SWE-bench](swe-bench-repository.md), Cybench) make it directly relevant to evaluating coding agents. Built on [Inspect](inspect-ai.md).
