# Observability for Generative AI and agentic AI systems

Microsoft Learn / Secure Future Initiative; AI-native logs/metrics/traces, OTel standardization

> **Excerpt from the original:** As enterprises adopt, build, and use AI systems—specifically Generative AI (GenAI) and agentic AI— traditional observability practices no longer suffice. Conventional software is largely deterministic, with predictable execution paths that operational telemetry can reliably explain. However, today’s AI systems don’t function this way—they are probabilistic by design. Outputs of GenAI systems vary across runs, and "execution" is a distribution over possible behaviors rather than a single reproducible path.
