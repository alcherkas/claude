# OpenSpec workflow extensions

Customizations layered on the OpenSpec spec-driven workflow.

## What was introduced

- **Deviations log** — created at apply, checked at verify
- **Design doc rules** — Design Decisions + Tradeoffs sections
- **Task manager sync** — apply mirrors tasks into Claude Code
- **Parallel/blocking tags** — task dependencies via blocks/blockedBy
- **Core/extended workflows** — Opus designs, Sonnet implements; run via `/openspec-core` or `/openspec-extended`
